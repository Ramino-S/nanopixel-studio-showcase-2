# 🎨 NanoPixel Studio — AI-Powered Creative Studio

> **NanoPixel Studio** — это современная веб-платформа для генерации и редактирования изображений на базе передовых нейросетей (**FLUX.2 Pro/Max** и новейшего семейства **Google Gemini 3.0** через **OpenRouter**). Проект построен по принципу Client-Side-First: все операции с ключами и хранением истории выполняются локально в браузере пользователя, что обеспечивает 100% конфиденциальность.

---

## 🚀 Стек технологий / Tech Stack

*   **Frontend Core:** React (Hooks, Context, Memoization), TypeScript (строгая типизация), Vite (быстрая сборка и HMR).
*   **Styling & UI:** Tailwind CSS, Lucide React (иконки), Glassmorphism дизайн-система с темной темой.

*   **AI Integrations:**
    *   Официальный SDK Google Gemini для работы с мультимодальными моделями семейства **Google Gemini** — включая Gemini 2.5 Flash, Gemini 3.0 Flash и флагманскую Gemini 3 Pro.
    *   **OpenRouter API** (Интеграция передовых моделей семейства **FLUX.2 Pro/Max** — Pro, Max, Flex, Klein).
 
*   **Deployment:** Cloudflare Pages (Wrangler deployment pipeline).

---

## ✨ Ключевые возможности / Core Features

### 1. 🎨 Генерация Изображений (Text-to-Image)
*   Генерация реалистичных и стилизованных изображений по текстовому описанию.
*   Тонкая настройка пропорций (1:1, 4:3, 16:9, 3:4, 9:16), качества (от Standard до 8K Masterpiece), стилей (Аниме, Киберпанк, Фотореализм и др.).
*   Мгновенная загрузка в форматах PNG и JPEG с авто-конвертацией через HTML5 Canvas.

### 2. 🖌️ Умный Редактор (Inpainting / Outpainting)
*   Локальный графический редактор с интерактивной кистью для маскирования областей.
*   Замена объектов на изображении по текстовому промпту (Inpainting).
*   Интеграция с ИИ для бесшовного дорисовывания краев и изменения деталей.

### 3. 👤 Портретный Адаптер (Portrait Generator)
*   Загрузка референсного фото лица.
*   Перенос черт лица на сгенерированные студийные портреты, бизнес-аватары или фэнтези-образы.
*   Сохранение естественной мимики, выбор эмоций, причесок и ракурсов.

### 4. 👥 Семейный Адаптер (Family Portrait)
*   Мультимодальная генерация с использованием двух референсных лиц (например, пары или родителей).
*   Создание красивых групповых портретов в различных стилях (Новогодние свитеры, Family Look, пляжный отдых, фэнтези).

### 5. 📊 Мониторинг Лимитов и Квот (Quota Control)
*   Интерактивный дашборд лимитов в реальном времени (RPM — запросы в минуту, RPD — запросы в сутки) для бесплатной сетки Gemini.
*   Умное каскадирование: автоматическое переключение на резервные ключи или платные модели FLUX.2 при наличии ключа OpenRouter.

### 6. 🛣️ Органайзер Инфографики 2.0 (Infographics - Coming Soon)
*   Стильный презентационный экран в разработке (Roadmap, интерактивная подписка на бета-тест, концепт-сетка безопасных зон под маркетплейсы WB, Ozon, Yandex).

## 📁 Структура проекта / Project Structure

```bash
nanopixel-studio/
├── components/              # React компоненты интерфейса
│   ├── ImageGenerator.tsx   # Модуль генерации изображений
│   ├── ImageEditor.tsx      # Встроенный графический редактор-кисть
│   ├── PortraitAdapter.tsx  # Модуль персональных портретов по фото
│   ├── FamilyAdapter.tsx    # Модуль парных/семейных генераций
│   ├── QuotaDisplay.tsx     # Дашборд лимитов и квот Gemini
│   ├── SettingsModal.tsx    # Настройки API-ключей OpenRouter/Gemini
│   └── LogViewer.tsx        # Встроенный отладчик и консоль событий
├── services/                # Бизнес-логика и API сервисы
│   ├── aiService.ts         # Оркестратор ИИ-моделей (маршрутизация)
│   ├── geminiService.ts     # Клиент работы с Google Gemini API
│   ├── openRouterService.ts # Клиент для генераторов семейства FLUX.2
│   └── historyService.ts    # Локальное хранилище и кэш генераций
├── types.ts                 # Глобальные интерфейсы и TypeScript перечисления
├── App.tsx                 # Главный компонент разводки вкладок приложения
├── index.css               # Базовые стили и интеграция Tailwind
├── index.html              # Точка входа HTML (с CDN загрузчиком)
└── tsconfig.json           # Конфигурация TypeScript
```

## 🔒 Безопасность и API-ключи

Приложение является **полностью бессерверным (Serverless)**. Все введенные в настройках ключи (OpenRouter, Google Gemini) сохраняются исключительно в вашем браузере (`localStorage`). Никакие ключи не передаются на сторонние бэкенды, данные идут напрямую в API провайдеров.
